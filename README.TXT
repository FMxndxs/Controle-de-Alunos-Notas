TITULO DO PROJETO: Cadastro de Alunos

OBJETIVO DO PROJETO: Criar um programa em linguagem Java para obter nome a partir de uma string, inverter nome, 
fazer nome bibliografico e realizar o cadastro de alunos

VERSÃO ou DATA: 30/04/2024

COMO INICIAR O PROJETO: Primeiramente criar uma classe Texto, para receber a string de texto, 
criar uma classe pessoa, para inserir informacoes como nome, idade, entre outras informacoes.
Criar uma classe Aluno para inserir dados cadastrais de um aluno e a partir disso, desenvolver outras classes para realizar 
o cadastro do aluno, inserir um novo aluno no cadastro ou remover algum aluno

AUTOR: Felipe Mendes Campos

INSTRUÇÕES PARA O USUÁRIO: Inicializar o programa atraves da classe Aplicacao e realizar o cadastro, inserindo informacoes como nome, 
idade, ra, semestre de estudo, quantidade de materias do aluno e as suas respectivas notas. O usuario pode utilizar do terminal de comando para 
realizar a criacao de nomes, disciplinas, notas, entre outros
